# 💧 Daily Water Intake Tracker

A **React.js web application** that helps users track their daily water consumption, manage entries, and calculate total intake between selected dates.

---

## 🚀 Features

* 🔐 User Authentication (Signup & Login)
* ➕ Add daily water intake
* ✏️ Edit existing entries
* ❌ Delete entries
* 📅 Filter water intake between two dates
* 📊 Calculate total water consumption
* 📄 Pagination for better data viewing
* 💾 Data persistence using **localStorage**
* 📱 Responsive UI with Bootstrap 4

---

## 🛠️ Technologies Used

* **React.js**
* **JavaScript (ES6)**
* **HTML5 & CSS3**
* **Bootstrap 4**
* **localStorage (Browser Storage)**

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/ShameerNavas/Daily-Water-Intake-Tracker.git
```

2. Navigate to the project folder

```
cd daily-water-intake-tracker
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm start
```

5. Open in browser

```
http://localhost:3000
```

---

## 📌 How It Works

* Users can **register and log in**
* Logged-in users can **add one water entry per day**
* All data is stored in **localStorage**
* Users can **edit, delete, and view entries**
* Date filter allows calculating **total water intake between two dates**
